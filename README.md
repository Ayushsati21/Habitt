Quick start

Requirements
- JDK 11 or 17 (recommended)
- Gradle (optional if using the Gradle wrapper included in the repo)

Build and test

```bash
# From the repository root (if Gradle wrapper exists):
./gradlew build

# Or with a system Gradle installation:
gradle build

# Run tests:
./gradlew test
```
